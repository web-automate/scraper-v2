Tech Stack

Engineered for Performance

Queue System

Leverages RabbitMQ to decouple API requests from heavy scraping tasks. This ensures the server remains non-blocking and scalable under load.

Type-Safe & Validated

Built with Node.js & TypeScript. All inputs are strictly validated using Zod, which auto-generates our OpenAPI/Swagger documentation.

Custom Infra & CDN

Hosted on VPS with Nginx Reverse Proxy. Includes a custom CDN for serving generated images and automated cleanup schedulers.

    headers: {
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_SECURE_KEY'

        topic: "Pola Pikir Generasi Z",
        keywords: ["Digital Native", "Mental Health"],

        webhookUrl: "https://your-callback.com/webhook",
        articleData: {

        }
    })
});

const result = await response.json();
